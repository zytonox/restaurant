## Features

- **Vue.js Framework** - Component-based architecture with Vue 3 Composition API
- **State Management** - Vuex for centralized cart and product state
- **Client-Side Routing** - Vue Router for seamless page navigation
- **Shopping Cart** - Full cart functionality with add/remove items
- **Modern CSS** - SCSS with BEM methodology for maintainable code

## Tech Stack

- Vue.js 3
- Vue Router
- Vuex
- SCSS/CSS3
- JavaScript (ES6+)
- Flexbox & CSS Grid
- BEM methodology
